## Buttons

- Primary: Lime bg + Ink text
- Secondary: Ghost/border

## Gradients

- Purple gradient: `135deg #5F2491 → #8E3AAC` (headers, heroes)

## Border Radius

- Fields: 8px
- Cards: 12px
- Buttons: 100px

## Rules

- NEVER use DM Sans
- NEVER use green #15803d
- Background charcoal: #1C1C1C (dark mode surfaces)
- Accent lime: #C8FF00 (CORTEX/dark contexts)
- All livoland.com frontend work must use these tokens
