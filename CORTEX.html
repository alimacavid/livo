Supabase connection · data persistence · configuration
Supabase Connection
Connect your Supabase project to persist Ideas across devices. Without this, data is stored in your browser only. Your Supabase URL and anon key are stored locally in this browser.
Create Supabase Table
Run this SQL in your Supabase project under SQL Editor to create the Ideas table.
CREATE TABLE cortex_ideas (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  content text NOT NULL,
  tags text[] DEFAULT '{}',
  created_at timestamptz DEFAULT now()
);

ALTER TABLE cortex_ideas ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public access" ON cortex_ideas FOR ALL USING (true);
