# LIVO Standard Operating Procedures

## General Job Protocol (All Divisions)
1. Confirm booking via Jobber 24hrs before
2. Send arrival window SMS to client (604 358 2358)
3. Arrive in uniform: LIVO purple shirt, cap
4. Walk-through with client before starting
5. Photo documentation: before and after every job
6. Collect payment on-site or via invoice
7. Request Google review before leaving

## Power Washing
1. Pre-rinse surface, identify stains
2. Apply detergent, dwell 5 min
3. Pressure wash low to high, maintain 12 inch distance
4. Rinse thoroughly, check no runoff to storm drain
5. Final inspection with client

## Moving
1. Protect all furniture with blankets before loading
2. Load heaviest items first against cab wall
3. Secure with ratchet straps every load
4. Walk-through destination before unloading
5. Place furniture per client instruction

## Carpet Cleaning
1. Pre-vacuum entire area
2. Pre-spray traffic lanes and stains
3. Dwell 10 min
4. Extract with Mytee, overlap passes by 50%
5. Groom pile, set fans, advise 4-6hr dry time

## Post-Construction Cleaning
1. Dust from ceiling down: light fixtures, vents, walls
2. Clean windows interior
3. Vacuum all surfaces
4. Mop hard floors
5. Final wipe-down of all surfaces
6. Photo documentation for builder sign-off

Update SOPs after any client complaint or near-miss. Version controlled in GitHub.
